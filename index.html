The Hoick Habit App

Helping you make new habits by not breaking the chain.





Track Action

Display a link icon for every day that a habit action is performed. Like Jerry Seinfeld you won't want to break the chain.


Web Server Backup

Send habit data to your own server. Know that the data is backed up to a place that isn't going to mine it for their own profit.


Native Reminders

Set reminders for habit actions using native apps and integration. Google Calendar on Android and the Reminders app on iOS.
